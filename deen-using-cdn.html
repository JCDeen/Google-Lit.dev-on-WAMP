So, to get this working, I had to:

  1. create _lit-test directory: D:\MyWampLocation\www\_lit-test>
  2. cd to D:\MyWampLocation\www\_lit-test>
  3. Run this cmd
    D:\MyWampLocation\www\_lit-test>npm i lit
  4. Started with the code shown below, as suggested on the linked page. Then added the obvious HTML which gives us the narrative (that your are reading):
        
	// Import lit-html
	import {html, render} from 'lit-html';

	// Define a template
	const myTemplate = (name) => html'<p>Hello ${name}</p>';

	// Render the template to the document
	render(myTemplate('World'), document.body);
